This site documents an operating system, constructed as a series of build steps that are mostly in order. I've been building these kinds of systems for years, using the general procedures found on Linux From Scratch. For a more recent OS version with dependency graph, see NðŸš«NIC.

You need to first bootstrap an LFS 9.0 os for this particular set of instructions.

I used this sequence to create 3SA. When I started, right around the time of the 9.0 release of LFS, emoji were not well supported for system work, and I needed the precision of LFS. As I come out the other side of this effort, the latest GNU/Linux and macOS deal with emoji just fine. I have not had the opportunity (or desire) to test on MS's V11, but the browser side works.
